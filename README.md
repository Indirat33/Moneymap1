# 💰 MoneyMap – Expense Management System

MoneyMap is a lightweight, PHP-based expense management system designed to help individuals and small teams track their income and expenses with ease. It provides both user and admin dashboards, making financial management organized and intuitive.

---

## ✨ Features

- 🧑‍💼 **User Dashboard** – Track income, expenses, and view summaries.
- 📊 **Admin Control Panel** – Manage users, view global stats, and ensure smooth operations.
- 💾 **MySQL Database Support** – Reliable, fast, and scalable storage.
- 🔒 **Authentication** – Secure login for users and admins.
- 📱 **Responsive UI** – Works seamlessly across desktop and mobile devices.

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Environment:** XAMPP (Apache & MySQL)

---
